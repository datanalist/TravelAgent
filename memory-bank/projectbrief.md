# Project Brief — TravelAgent

## Суть проекта

**TravelAgent** — мультиагентный AI-консьерж для туроператоров и турагентств, ориентированный на **high-end сегмент**. Система автоматизирует первичные продажи и сервис через Telegram Bot и Web-чат.

## Проблема

Туроператоры теряют конверсию на этапе первичной коммуникации: менеджеры не успевают реагировать, не персонализируют предложения. В premium-сегменте ожидания — 24/7, экспертный тон, персонализация.

## Цели

| Цель | Измеримый результат |
|---|---|
| Квалификация лидов | ≥ 15% конверсии (vs ~5% baseline) |
| Заполненный профиль | ≥ 70% лидов |
| Скорость первого ответа | < 5 секунд |
| High-end тон | ≥ 75% оценок «соответствует» |
| Кросс-канальная непрерывность | Сессия сохраняется TG ↔ Web |

## Скоуп PoC / Milestone 1

**В скоупе:**
- Telegram Bot + Web SSE чат на едином backend
- Квалификация лида, сбор профиля
- Поиск туров (`search_tours` — заглушка/простая БД)
- Создание лида в CRM (таблица `leads`)
- Streaming ответов (SSE для Web)
- Двухуровневая память (Redis + PostgreSQL)

**Вне скоупа:**
- Реальные CRM/агрегаторы туров/API авиакомпаний
- Оплата и бронирование
- Голос, мультиязычность (только русский)
- Авторизация с историей между сессиями (базовая сессия)
- Мобильное приложение, A/B-тестирование
